#Face Recognition System 🎯

## 📌 Description

This project detects and recognizes human faces in real-time using a webcam.

## Features

* Real-time face detection
* Face recognition using trained images
* Unknown face detection

## 🛠️ Technologies Used

* Python
* OpenCV
* face-recognition
* NumPy

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python face_recognition.py

## 📁 Project Structure

* images/ → dataset
* face_recognition.py → main code
